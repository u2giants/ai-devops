### STEP 1 — INSTALL WINDSURF

1. Go to https://windsurf.com and download the Windows installer.
2. Run it, accept all defaults.

**Verify:** Open Windsurf — a code editor window appears.

---

### STEP 2 — INSTALL ROO CODE EXTENSION IN WINDSURF

1. Open Windsurf.
2. Click the **Extensions** icon in the left sidebar (4 squares).
3. Search for `Roo Code`.
4. Install "Roo Code" by Roo Veterinary Inc.

**Verify:** A paw print icon appears in the left sidebar.

---

### STEP 3 — GET AN OPENROUTER API KEY

1. Go to https://openrouter.ai and sign in.
2. Click your profile → **API Keys** → **Create Key** (name it "Roo Code").
3. Copy the key — you'll need it in Step 4.

---

### STEP 4 — CONFIGURE ROO CODE WITH OPENROUTER

1. In Windsurf, click the Roo Code paw icon.
2. Click **Settings** (gear icon).
3. Under **API Provider**, select **OpenRouter**.
4. Paste the API key from Step 3 and click **Save**.
