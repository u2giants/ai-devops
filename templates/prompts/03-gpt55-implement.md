# Stage 03 — Implementation (GPT-5.5 / Codex)

**Model role:** GPT-5.5 / Codex is the implementation model.

## Your task

Implement the **approved** plan — nothing more.

Rules:

- Implement **only** what the approved plan describes. Do not add scope.
- Make the **smallest safe change** that satisfies the plan.
- Do **not** refactor unrelated code, rename unrelated symbols, or reformat
  files you did not otherwise need to touch.
- **Add or update tests** for the behavior you change.
- Match the surrounding code's style, naming, and conventions.
- If you discover the plan is wrong or incomplete, stop and report the mismatch
  rather than improvising a large change.

## When done

Summarize:

- Which files you changed and why.
- Which tests you added or updated.
- Anything you intentionally did **not** do (and why).
- Any deviation from the plan and the reason.
